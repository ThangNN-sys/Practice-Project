JavaScript (JS)

* Chức năng:

- Tương tác người dùng: JavaScript giúp trang web phản hồi lại 
các thao tác của người dùng như nhấn nút, điền biểu mẫu, cuộn trang.

- Xử lý logic: Thực hiện các logic phía frontend như 
xác minh dữ liệu nhập, điều hướng, và tính toán.

- Tạo hiệu ứng động: Tạo các hiệu ứng chuyển động phức tạp 
và thực hiện các thay đổi trực tiếp trên HTML hoặc CSS mà không cần tải lại trang.

* Trình tự xây dựng JavaScript:

- Thiết lập các sự kiện: Đặt các sự kiện cho các phần tử HTML như 
click, submit, hover, focus.

- Xử lý sự kiện: Tạo các hàm JavaScript để xử lý sự kiện, 
thực hiện các hành động tương ứng như kiểm tra dữ liệu nhập vào, thay đổi style, hoặc gọi API.

- Thực hiện logic frontend: Xây dựng các logic liên quan đến ứng dụng như 
tính toán, điều hướng, và quản lý trạng thái.

- Thêm hiệu ứng động: Sử dụng JavaScript để thay đổi trực tiếp nội dung của trang 
hoặc tạo các hiệu ứng đặc biệt.
