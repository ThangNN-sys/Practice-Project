CSS (Cascading Style Sheets)

* Chức năng:

- Định dạng và trang trí: CSS được sử dụng để thay đổi kiểu dáng của các thành phần HTML, 
bao gồm màu sắc, phông chữ, kích thước, khoảng cách, và bố cục.

- Bố cục: Sử dụng CSS để điều chỉnh cách các thành phần sắp xếp trên trang, ví dụ như 
hiển thị theo dạng lưới (grid) hay dạng khối (flexbox).

- Hiệu ứng và tương tác: Tạo các hiệu ứng chuyển động (animation) 
và hiệu ứng hover khi người dùng tương tác với các phần tử.

* Trình tự xây dựng CSS:

- Xác định style chung: Bắt đầu bằng việc thiết lập các style chung cho toàn bộ trang, 
như phông chữ, màu nền, và cách căn lề mặc định.

- Thiết kế layout: Sử dụng CSS để tạo ra layout tổng thể, phân chia các khu vực chính 
và sắp xếp chúng bằng cách sử dụng grid hoặc flexbox.

- Định dạng chi tiết cho từng phần tử: Áp dụng style cụ thể cho từng phần tử HTML 
như tiêu đề, đoạn văn, nút bấm.

- Thêm hiệu ứng: Áp dụng các hiệu ứng khi người dùng hover, nhấn chuột, 
hoặc các hiệu ứng động khác.
