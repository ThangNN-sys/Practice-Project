* Vai trò:

Resources chứa các file cấu hình, tài nguyên tĩnh như file
application.properties, logback.xml, SQL scripts, các tệp
tài nguyên tĩnh (ảnh, file JSON, etc.).

* Cách hoạt động:

Khi ứng dụng chạy, các file cấu hình trong resources sẽ được đọc
để cấu hình hệ thống, như cấu hình kết nối database, cài đặt logging, v.v.
